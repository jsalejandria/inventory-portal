# Project Overview

## Description
A simple inventory application

## Features
> Create and product entry with the following information:
  - Product name
  - Unit of Measure
  - Price
  - Expiry Date
  - Stocks on Hand
  - Total Value of Inventory
> Edit details of an existing product entry
> Delete product entry

## Frameworks
- Web framework: React
- CSS framework: Bootstrap
