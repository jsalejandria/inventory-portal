# Project Overview

## Description
A simple inventory application

## Features
1. Create and product entry with the following information:
  - Product name
  - Unit of Measure
  - Price
  - Expiry Date
  - Stocks on Hand
  - Total Value of Inventory
2. Edit details of an existing product entry
3. Delete product entry

## Frameworks
- Web framework: React
- CSS framework: Bootstrap
